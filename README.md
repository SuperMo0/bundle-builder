## Testing

![CI](https://github.com/SuperMo0/bundle-builder/actions/workflows/ci.yml/badge.svg)

Run the test suite:

```bash
npm test
```

Run it with a coverage report:

```bash
npm run test:coverage
```

| ---------- | -------- |
